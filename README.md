# object-to-argv

[![Build Status](https://travis-ci.org/evanlucas/object-to-argv.svg)](https://travis-ci.org/evanlucas/object-to-argv)

Convert an object to an array of arguments to pass to a cli process

## Install

```bash
$ npm install object-to-argv
```
